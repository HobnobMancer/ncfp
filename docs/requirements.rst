============
Requirements
============

To use ``ncfp`` you will need:

Operating System
    - macOS or Linux

Python
    - Python version 3.5 or higher

Python Packages
    - `biopython`_
    - `tqdm`_


.. _biopython: http://biopython.org/
.. _tqdm: https://github.com/noamraph/tqdm
